Repositorio creado para subir proyectos relacionados con un curso de Sence 

Se agregó el archivo app.js con la funcionalidad de scroll suave


Ejemplos de comandos de git

git init

Definición: Inicializa un repositorio Git en la carpeta actual.

Ejemplo:

git init
git add

Definición: Agrega archivos o cambios al área de preparación (staging).

Ejemplo:

git add .
git commit

Definición: Guarda los cambios realizados en el repositorio con un mensaje descriptivo.

Ejemplo:

git commit -m "Primer commit"
git status

Definición: Muestra el estado actual del repositorio y los archivos modificados.

Ejemplo:

git status
